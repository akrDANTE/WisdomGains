# SQL query language
SQL is a query language(query language is a way to interact with structured data) which is the standard for interacting with relational databases. Relational databases are table based which contain named columns as schema, and data as rows, these tables can be related to one another by foreign keys, that is one table may contain references to other table and also have integrity constariants like UNIQUE,NOT NULL etc enforced while having ACID property which gaurantees transactions(All or None). 
SQL allows to fetch data, insert data, update data, delete data, create tables, perform joins on multiple tables.. etc. 
There are many databases which use SQL language like SQLite, MySQL, PostgreSQL etc.
SQL is declarative(it says what should happen without caring about how) set-based language that is it performs operations on mathematical sets like intersection, union etc.
